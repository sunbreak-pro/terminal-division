# Solutions

This directory contains documentation for resolved bugs and their fixes.

## Purpose

- Document the root cause of bugs
- Explain the fix implementation
- Serve as reference for similar issues in the future

## Naming Convention

Use `{problem-description}-fix.md` format.

Examples:
- `packaged-app-japanese-garbled-fix.md`
- `cmd-backspace-undo-history-fix.md`

## Template

```markdown
# Fix: [Problem Description]

## Problem
Brief description of the issue.

## Root Cause
Why was this happening?

## Solution
How was it fixed?

## Files Changed
- path/to/file1.ts
- path/to/file2.ts

## Related
- Link to bug report
- Related issues
```
