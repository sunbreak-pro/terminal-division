# Claude Configuration

このディレクトリは Claude Code との連携に関する設定やドキュメントを管理します。

## ディレクトリ構造

- `settings.local.json` - ローカル設定ファイル
- `bug/` - バグ追跡関連
- `plans/` - 機能計画・アイデア管理

## plans/ ディレクトリについて

`plans/` ディレクトリは、今後追加したい機能やアイデアを整理・記録するための場所です。

### 用途
- 新機能の企画書・設計メモ
- UI/UX 改善案
- リファクタリング計画
- 技術的な検討事項

### ファイル命名規則
- `{カテゴリ}_plan.md` 形式を推奨（例: `UI_plan.md`, `feature_plan.md`）
