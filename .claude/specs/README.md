# Specifications

This directory contains design specifications and requirements documentation.

## Structure

- `templates/` - Reusable templates for creating specs
- Feature specs are placed directly in this directory

## Creating a New Spec

1. Copy the appropriate template from `templates/`
2. Name the file descriptively: `{feature-name}-spec.md`
3. Fill in all sections of the template

## Templates

- `feature-spec.md` - For new feature proposals
- `design-decision.md` - For architectural decisions (ADR format)
